# 图片资源说明

请将以下图片文件放置在此目录：

## result-ui.png
- **用途**：结果页 UI 框架图片
- **格式**：PNG（支持透明）或 JPG
- **建议分辨率**：1920x1080 或更高
- **建议文件大小**：< 1MB
- **设计建议**：
  - 在图片上预留清晰的内容区域
  - 原始图片区域（约 30% 宽度）
  - 变老图片轮播区域（约 55% 宽度）
  - 三个文字区域（姻缘、事业、财运，各约 28% 宽度）
  - 可以包含装饰性元素和品牌 LOGO

## 布局参考
默认布局配置（可在 `js/config.js` 中调整）：
- 原始图片：top: 10%, left: 5%, width: 30%, height: 40%
- 变老图片轮播：top: 10%, left: 40%, width: 55%, height: 40%
- 姻缘文字：top: 55%, left: 5%, width: 28%
- 事业文字：top: 55%, left: 36%, width: 28%
- 财运文字：top: 55%, left: 67%, width: 28%

## 注意事项
- 图片需要经过压缩优化
- 建议使用高质量但文件大小合理的图片
- 确保图片在不同屏幕尺寸下都能正常显示
