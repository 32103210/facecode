# 视频资源说明

请将以下视频文件放置在此目录：

## background.mp4
- **用途**：首页背景视频
- **格式**：MP4（H.264 编码）
- **比例**：16:9
- **建议分辨率**：1920x1080 或 1280x720
- **建议文件大小**：< 5MB
- **时长**：10-30 秒循环播放

## loading.mp4
- **用途**：加载页预加载视频
- **格式**：MP4（H.264 编码）
- **建议分辨率**：1920x1080 或 1280x720
- **建议文件大小**：< 10MB
- **时长**：5-15 秒（会循环播放直到 API 返回）

## 注意事项
- 视频文件需要经过压缩优化以加快加载速度
- 建议使用 H.264 编码以确保浏览器兼容性
- 可以使用 FFmpeg 等工具进行视频压缩和转换
